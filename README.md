# 4-peace-linebot
LINE Bot for 4-Peace - 八色禪卡線上抽牌 (8張禪卡 + 每日日記 + 月度统計)

# 檔案結構
zen-card-linebot/
├── index.js          # 主程式
├── package.json      # 專案設定檔
├── cards.json        # 禪卡資料
├── database.js       # 資料庫操作
├── .env             # 環境變數（存放密鑰）
├── .gitignore       # Git 忽略清單
├── README.md        # 說明文件
└── public/          # 靜態檔案資料夾
    └── index.html   # 簡單的網頁後台
